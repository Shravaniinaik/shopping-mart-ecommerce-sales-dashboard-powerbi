🛒 Shopping Mart E-Commerce Sales Dashboard

📖 Overview

The Shopping Mart E-Commerce Sales Dashboard is an interactive Business Intelligence solution developed using Power BI. This dashboard transforms raw e-commerce sales data into meaningful insights, enabling users to analyze sales performance, profitability, customer behavior, product categories, payment methods, and regional trends.

Designed with dynamic filters and interactive visualizations, the dashboard helps stakeholders make informed, data-driven business decisions.

🚀 Project Highlights

* Interactive Power BI Dashboard
* Dynamic Filtering and Cross-Filtering
* Sales and Profit Analysis
* Customer Insights
* Regional Performance Tracking
* Category and Sub-Category Analysis
* Payment Mode Analysis
* Business KPI Monitoring

📊 Dashboard Features

Key Performance Indicators (KPIs)

* Total Sales Amount
* Total Quantity Sold
* Total Profit
* Average Order Value (AOV)

Visualizations

* Profit by Month
* Sales Amount by State
* Quantity by Category
* Quantity by Payment Mode
* Profit by Sub-Category
* Customer-wise Sales Analysis

Interactive Elements

* Quarter-wise Filters
* State-wise Filters
* Dynamic Cross-Filtering Across Visuals
* Interactive Data Exploration
